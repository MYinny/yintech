# YIN TECH Website

Static website for YIN TECH, a small-business managed IT/MSP company.

## Structure

```text
.
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── assets/
    ├── logos/
    └── images/
```

## Before publishing

Update these placeholders in `index.html`:

- `hello@yintech.ca`
- phone number link/text
- service area
- FAQ answer about remote/onsite support
- any final service/package wording

## GitHub upload

You can upload this folder directly to GitHub.

If using GitHub Pages:

1. Push these files to a repo.
2. Go to repo Settings → Pages.
3. Set source to your main branch root.
4. GitHub will publish `index.html`.

No build step is required.
